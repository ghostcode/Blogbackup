Mac终端变主题

最近看终端眼睛有点累，想改变一下主题。

如果不想下新主题,终端默认有很多主题：

  1. 先打开终端：终端—偏好设置—设置

  2. 选好之后记得点击默认，接着重启终端。

我现在的主题是：solarized

下载：

$ git clone git://github.com/altercation/solarized.git

其实有两个主题：Dark和light。

下好之后双击你需要的就会自动安装，重复上面的步骤。
