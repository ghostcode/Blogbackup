Javascript算法-排序

笔试会遇到一些算法问题，为了不被虐，也是为了增加知识，开始一个系列的算法学习。

交换排序的基本思想是：两两比较待排序记录的关键字，发现两个记录的次序相反时即进行交换，直到没有反序的记录为止。

冒泡排序

调用：

sort([119,20,10,3]) //[3,10,20,119]

sort(123) //123 is not an Array.
