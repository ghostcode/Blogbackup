Using flexbox helps achieve layouts that used to be more difficult to create. Flexbox rules define the placement of elements contained inside a parent that has the display: flex property. See Can I Use for browser support (it's pretty high). See CSS-Tricks for more thorough descriptions of flexbox rules.

Flexbox layout is most appropriate to the components of an application, and small-scale layouts, while the Grid layout is intended for larger scale layouts.
